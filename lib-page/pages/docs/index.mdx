# Getting Started

Add beautiful notifications to your React app with [every-notify](https://github.com/timolins/react-hot-toast).

### Install with Yarn

```sh
yarn add every-notify
```

### Install with NPM

```sh
npm install every-notify
```

## Basic usage

```jsx
import toast, { Toaster } from 'every-notify';

const notify = () => toast('Here is your toast.');

const App = () => {
  return (
    <div>
      <button onClick={notify}>Make me a toast</button>
      <Toaster />
    </div>
  );
};
```
